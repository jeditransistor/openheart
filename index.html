Live demo

Openheart uses Socket.IO and Font Awesome to give a 1bpm signal to client-side JS. Signal time can be adjusted in the setTimeout(). Notice that the client JS has a .5 second delay; this is to allow some time for the heart to "animate"; ALL time-keeping is done on the server.

server.js

setInterval(function() {
    io.sockets.emit('heartbeat', "fa fa-heart fa-stack-1x fa-inverse");
}, 1000);

client.js

var socket = io();
socket.on('heartbeat', function(msg){
      document.getElementById("heart").className = "fa fa-heart fa-stack-1x fa-inverse";
      setTimeout(function(){ document.getElementById("heart").className = " "; }, 100);
});
